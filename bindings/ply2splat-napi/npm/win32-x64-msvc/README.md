# @ply2splat/node-win32-x64-msvc

This is the Windows x64 native binary for `@ply2splat/node`.

This package is installed automatically as an optional dependency of `@ply2splat/node` on the appropriate platform.

## Installation

You should install the main package instead:

```bash
npm install @ply2splat/node
```
